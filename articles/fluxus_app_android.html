Cos'è Fluxus? Descrizione e Features

Fluxus è un'App per sistemi operativi mobile Android che permette di ternere traccia del denaro tramite l'aggiunda di ogni singola spesa/guadagno.

  • Interfaccia utente intuitiva e facile da usare
  • Aggiungere nuovi record con un semplice tap
  • Gestione di diversi conti (Contanti, banca, carte...)
  • Modalità spese previste che ricoda tramite una notifica di effettuare un pagamento
   

Tecnologie utilizzate

  • Java
  • SQLite
  • Object Oriented Programming
  • Android Studio
